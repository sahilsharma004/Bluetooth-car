# Bluetooth-car

ESP32 Bluetooth Controlled Car

This project is a Bluetooth-controlled robotic car built using an ESP32, an L298N/L293D motor driver, and DC motors. The car can be controlled using the "Arduino Car" Android app (available on Play Store), which sends serial commands via Bluetooth.

You are using the Arduino Car App for remote control. The app sends the following Bluetooth commands:

Command	Action
F	Move Forward
B	Move Backward
L	Turn Left
R	Turn Right
S	Stop

Make sure Bluetooth is turned ON and connect to the device name:
